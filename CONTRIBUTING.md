# Contribuciones

Este proyecto es educativo y forma parte de la asignatura de Ingeniería de Datos.

## Cómo Contribuir

Si encuentras errores o tienes mejoras:

1. **Fork** el repositorio
2. Crea una rama con tu feature: `git checkout -b feature/nombre-mejora`
3. Commit tus cambios: `git commit -m "Descripción clara del cambio"`
4. Push a la rama: `git push origin feature/nombre-mejora`
5. Abre un Pull Request

## Normas de Código

- **SQL:** Usar mayúsculas para palabras clave, comentarios en español
- **Python:** Seguir PEP 8
- **Commits:** Mensajes claros y descriptivos en español

## Reportar Errores

Usa la sección de Issues con:
- Descripción clara del problema
- Pasos para reproducir
- Resultado esperado vs actual
